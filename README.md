# DM-All-Server
1. Go here https://discord.com/developers/applications and create a bot!

2. Then go to the bot section and check the last three features
<img width="1919" height="1079" alt="Στιγμιότυπο οθόνης 2025-11-02 101558" src="https://github.com/user-attachments/assets/db0e33a6-5494-4c9b-b87b-5cdd7e1ef4e5" />

3. Change the TOKEN, GUILD_ID, COOLDOWN in the dm_bot.py file

4. Put the bot to the server by going to the OAuth2 section and select: Bot and Administrator. Copy link and put it to the server
<img width="1919" height="1079" alt="Στιγμιότυπο οθόνης 2025-11-02 102321" src="https://github.com/user-attachments/assets/7949a837-b4a1-4cdb-aeec-501d7caf29a9" />

5. Run this command: "pip install -r requirements.txt" in cmd of the file

6. And run the bot in cmd with: python dm_bot.py
